# blincus config set

Save a value in the config file

| Attributes       | &nbsp;
|------------------|-------------
| Alias:           | s

## Usage

```bash
blincus config set KEY VALUE
```

## Examples

```bash
blincus set engine lxd
```

```bash
blincus set fedora.latest 40
```

## Arguments

#### *KEY*

Config key

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *VALUE*

Config value

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
